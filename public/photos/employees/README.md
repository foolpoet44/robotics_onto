# 개인별 프로필 사진

역량 페이지(개인별 역량정보)의 성명 좌측 아바타에 사용되는 사진 디렉터리입니다.

## 사용 방법

- 파일명을 **사번(employeeId)** 으로 지정합니다. 예: 사번이 `A12345` 인 직원은 `A12345.jpg`.
- 형식은 `.jpg` 를 사용합니다 (`EmployeeCompetencyDashboard` 의 경로 규칙: `/photos/employees/{employeeId}.jpg`).
- 정사각형(예: 200×200) 이미지를 권장합니다. 원형으로 잘려 표시되며 가운데 정렬(object-fit: cover)됩니다.

## 폴백 동작

해당 사번의 사진 파일이 없으면, 코드/데이터 수정 없이 자동으로
**이름 이니셜 + 전문영역 색상** 아바타가 표시됩니다. 사진을 추가하면 즉시 반영됩니다.
